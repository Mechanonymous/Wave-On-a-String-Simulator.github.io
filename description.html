About the Simulation

This interactive simulation demonstrates the behavior of a transverse wave traveling along a string, based on the fundamental principles of the wave equation. Users can visualize how disturbances propagate through a medium, where each point on the string oscillates while energy travels along its length. ∂2y/∂t2 = c2( ∂2y/∂x2 )

The simulation models the relationship between displacement, time, and position using the wave function y(x,t), allowing users to observe how waves form, move, and interact. By adjusting parameters such as amplitude, frequency, pulse width, and damping, users can explore how different conditions affect wave motion.

It also includes boundary conditions such as a fixed end, where waves reflect and invert, demonstrating real physical behavior. As waves travel and overlap, the simulation showcases interference patterns, illustrating both constructive and destructive superposition. Over time, these interactions can form standing waves, with visible nodes and antinodes.

This tool provides a dynamic way to understand how waves carry energy without transporting matter, and how properties like tension and mass influence wave behavior. The simulation is inspired by the interactive learning tools developed by PhET Interactive Simulations.
